# Convert a single pdf to multiple pngs one per page

Convert a single pdf to multiple pngs one per page

## Usage

``` r
.convert_pdf_to_pngs(pdfFile)
```

## Arguments

- pdfFile:

  an input pdf file

## Value

a list of png files one per page.
